ประวัติส่วนตัว
ชื่อ: นาย ชญานินท์ จีนขจร ชื่อเล่น นักเก็ต เกิดวันที่ 12 มิถุนายน 2543 อายุ 21 ปี
การศึกษา: มหาวิทยาลัยเทคโนโลยีราชมงคลธัญบุรี สาขา เทคโนโลยีสารสนเทศการศึกษา คณะ ครุศาสตร์อุตสหกรรม
ประสบการณ์การทำงาน: ฝึกงานที่เทศบาลนครรังสิต ตำแหน่ง ผู้ช่วยนักวิชาการคอมพิวเตอร์
รายละเอียดงาน: แก้ไขปัญหาคอมพิวเตอร์เบื้องต้น ติดตั้งและลงโปรแกรมที่ผู้ใช้ต้องการ ดูแลเว็บไซต์เทศบาลและลงข่าวในเว็บไซต์
สร้างหน้าเว็บไซต์เพื่อให้ประชาชนสามารถตรวจสอบและดูได้ว่าพนักงานในเทศบาลคนไหนอยู่ตำแหน่งไหน
ความสามรถ: HTML, Css, java, python
โปรแกรมที่ใช้ได้: adobe photoshop cs6, Visual Studio Code, pycharm, microsoft
